1)Интеграционный тест предназначен только для postman(для другого я не пробовал)
2)Для успешного выполнения тесты нужно запустить в Run folder, после чего приложить файл json, нажав на кнопку select file, по умолчанию будет стоять 9 итераций
3)api должны быть строго в последовательности signup->login->refresh_token->logout(так как json написан только для такой последовательности)
4)По итогу должно быть 4 правильных теста и остальные неправильные, неправильные тесты сделаны намеренно, чтобы проверить как будет себя вести программа
